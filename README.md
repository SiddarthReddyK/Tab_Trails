# Tab Trails 🕹️

A pixel-themed Chrome extension that helps students and researchers track their browsing sessions as a visual trail.

## What it does
- Start a named research session called a "Trail"
- Automatically captures every tab you open
- Displays tabs as a vertical timeline inside the popup
- Add notes to each tab to remember why you saved it
- Delete tabs you don't need
- Stop the trail when you're done

## How to install
1. Download or clone this repo
2. Go to `chrome://extensions` in Chrome
3. Turn on **Developer Mode** (top right)
4. Click **Load Unpacked**
5. Select the `tab-trails` folder

## Built with
- HTML, CSS, JavaScript
- Chrome Extensions API (Manifest V3)

## Why I built this
You know that feeling — you're deep in a rabbithole, trying to learn something, and suddenly you have 20 tabs open with zero memory of why you opened half of them. Tab Trails was built for that exact moment. It keeps track of your journey so your future self knows what past you was thinking.
